This is Automation Script to Install some of Pentesting Tools

What this script do ?

Install :
- Go [latest]
- Nuclei
- Subfinder
- Httpx
- Interactsh
- Dalfox
- Ffuf
- Qsreplace
- Anew
- Waybackurls
- Kxss
- Gf
- Gospider
- Gau
- GetJS
- Unfurl

Usage : `./main.sh` or `sudo ./main.sh`

Note : Add permission for execute `chmod +x ./main.sh ./checkgo.sh`
