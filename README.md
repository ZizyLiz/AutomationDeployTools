This is Automation Script to Install some of Pentesting Tools for Linux

What this script do ?

Install :
- Go [latest]
- Nuclei
- Subfinder
- Httpx
- Interactsh
- Dalfox
- Ffuf
- Qsreplace
- Anew
- Waybackurls
- Kxss
- Gf
- Gospider
- Gau
- GetJS
- Unfurl
- Cidrex

Usage : `./main.sh` or `sudo ./main.sh`
